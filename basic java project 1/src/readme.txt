
Q1: The code is for converting the money from one unit to another. There are 12
different conversions that can be done which are tl2usd, usd2tl, tl2euro, euro2tl,
euro2usd, usd2euro, gbt2tl, tl2gbp, euro2gbp, gbp2euro, gbp2usd, and usd2gbp. There
is also an input for exitting which is q. When it is run the program, it asks the
user to give an input as what it is wanted to convert. The input should be one of
the 13 possible inputs. If it is not, it asks the same question again. After the
right choice, it asks the unit value for the type that is wanted to be transferred
to. Then it asks the user to give amount money in the type that is wanted to be
transferred from as an input. Finally, it gives the converted value as an output and
go back to the menu. To exit the program, it is needed to write 'q' to the question
'What do you want to convert?'. The numbers cannot be 0 or negative. For giving a float
as an input, it is needed to give them in the format '3,4', not the format '3.4'.

Q2: This program is for the solution of the equation ax + b = 0. It asks the user to give
the a and b values. After that, it finds the x value and shows it as an output. It also
shows if x is undefined or undeterminate.

Q3: This program is for finding the weakest person in weight. It asks the user to enter the
weight of the person and then the full name of the person. It asks these 2 questions
continuously until the user enters -1 to the weight. Finally, it shows the first name of the
weakest person and his/her weight. If there is no person in input, it shows there is no
person to name.
