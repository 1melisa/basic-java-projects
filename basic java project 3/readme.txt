1: There is a class Circle2D which is for creating 2 dimensional circles. There are some functions that are getArea,
getPerimeter, overlaps and 2 different contains. getArea finds the area of the circle. getPerimeter finds the perimeter
of the circle. overlaps checks whether the circle overlaps with the specified circle or not. contains which is with a
point parameter checks whether the circle contains a specified point or not. contains which is with a circle parameter
checks whether the circle contains a specified circle or not. To run the code, it is needed to change the name of the
file "1.1.java" as "Circle2D.java" and the name of the file "1.java" as "Q1.java".

2: There is a class MyInteger which is for creating a number. There are some functions that are 3 different isEven,
isOdd and isPrime, and 2 different equals and parseInt. isEven with no parameter checks whether the value is even or not,
while isEven with the integer parameter checks whether the specified value is even or not and isEven with the object
parameter checks whether the value of the specified object is even or not. isOdd with no parameter checks whether the
value is odd or not, while isOdd with the integer parameter checks whether the specified value is odd or not and isOdd
with the object parameter checks whether the value of the specified object is odd or not. isPrime with no parameter
checks whether the value is prime or not, while isPrime with the integer parameter checks whether the specified value is
prime or not and isPrime with the object parameter checks whether the value of the specified object is prime or not.
equals with the integer parameter checks whether the specified value equals to the value or not while equals with the
object parameter checks whether the value of the specified object equals to the value or not. parseInt with the char[]
parameter changes the array to the integer while parseInt with the String parameter changes the string to the integer.
To run the code, it is needed to change the name of the file "2.1.java" as "MyInteger.java" and the name of the file
"2.java" as "Q2.java".
